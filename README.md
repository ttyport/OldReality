# OldReality
Collection of retro arcade games

# Launch and install

Make sure that [Python3](https://python.org) is installed

Installing dependencies:
```
pip3 install -r requirements.txt
```

### Launch

Windows, macOS:
```
python main.py
```

Linux:

```
sudo sh install.sh  # Installing game
```

### Deletion

Linux:

```
sudo sh uninstall.sh
```
